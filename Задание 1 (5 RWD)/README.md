# Отель "Комфорт" - Веб-приложение для автоматизации деятельности

Это адаптивный веб-интерфейс системы автоматизации деятельности отеля, разработанный с использованием современных принципов адаптивной и отзывчивой верстки.

## Особенности проекта

- **Mobile-first подход**: Верстка сначала разработана для мобильных устройств, а затем адаптирована для больших экранов
- **Адаптивный дизайн**: Страница корректно отображается на экранах от 320px до 2560px
- **Медиа-запросы**: Используются для создания нескольких точек перехода
- **Методология BEM**: Применяется для именования классов
- **SCSS/CSS**: Использование переменных, миксинов и вложенной структуры
- **Flexbox и Grid**: Применяются для современной и гибкой компоновки элементов
- **Адаптивные изображения**: Используются srcset с дескрипторами w и тег picture
- **Доступность**: Учтены основные принципы веб-доступности (WCAG)

## Особенности адаптивной верстки

### Метатег viewport

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Медиа-запросы

В проекте используются следующие точки перехода:
- 576px (sm): Маленькие устройства (телефоны)
- 768px (md): Средние устройства (планшеты)
- 992px (lg): Большие устройства (десктопы)
- 1200px (xl): Очень большие устройства
- 1400px (xxl): Экстра большие устройства

### Адаптивные изображения

Используется атрибут `srcset` с дескриптором `w`:

```html
<img src="img/room1-medium.jpg" 
     srcset="img/room1-small.jpg 400w, img/room1-medium.jpg 800w, img/room1-large.jpg 1200w"
     sizes="(max-width: 600px) 100vw, (max-width: 1024px) 50vw, 33vw"
     alt="Стандартный номер">
```

Также используется тег `picture` для обеспечения поддержки современных форматов изображений:

```html
<picture>
  <source srcset="img/gallery1-small.webp 400w, img/gallery1-medium.webp 800w" type="image/webp">
  <img src="img/gallery1-medium.jpg" 
       srcset="img/gallery1-small.jpg 400w, img/gallery1-medium.jpg 800w"
       sizes="(max-width: 600px) 100vw, (max-width: 1024px) 50vw, 33vw"
       alt="Лобби отеля">
</picture>
```

### Шаблоны поведения макета

В проекте используются различные шаблоны компоновки:
- **Mostly Fluid**: Основной контент перестраивается в зависимости от ширины экрана
- **Column Drop**: Колонки перестраиваются вертикально на маленьких экранах
- **Layout Shifter**: Принципиально разные макеты для разных размеров экрана

## Методология BEM

Примеры использования методологии BEM (Block, Element, Modifier):
- Блок: `.header`, `.room-card`, `.booking-form`
- Элемент: `.header__logo`, `.room-card__title`, `.booking-form__submit`
- Модификатор: `.hero__button--secondary`, `.form-group--full`

2. Откройте файл `index.html` в браузере для просмотра

## Компиляция SCSS (при необходимости)

Если вы хотите внести изменения в SCSS файлы, вам нужно скомпилировать их в CSS:

```bash
# С использованием node-sass
npm install -g node-sass
node-sass scss/styles.scss css/styles.css --output-style compressed
```
